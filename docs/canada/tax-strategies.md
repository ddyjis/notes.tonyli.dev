# Tax Strategy

## Tax Free Savings Account

- Contribution room 之內所有收入免稅；入多過 contribution room 的話，多出嚟嘅錢會罰
- 因為入落去嘅錢係稅後，所以唔會扣稅，同樣攞返出嚟都唔會計收入
- 買咗唔合資格或禁止交易嘅資產亦會計稅
- 如果稅務年度內成為 non-resident，係唔可以做 contribution，否則會計稅
- 一年內嘅提款會加入下年度嘅 contribution room

## Registered Retirement Savings Plans

Contribution limit is 18% of the earned income, upper limit is $29210 (2022)

- 戶口入面賺到嘅錢唔會計稅
- 提款會計入當年收入
- 未用完嘅 contribution room 會計落下一年
- 有分
  - 個人
  - 伴侶
  - 群組（Group RRSP）（老闆整嘅）
  - 集資（Pooled RRSP）（小企業用）
- 到 71 歲時，RRSP 就要提取或轉入 RRIF ( #todo : find out more ) 或放入年金
- [Registered Retirement Savings Plan (RRSP) - Canada.ca](https://www.canada.ca/en/revenue-agency/services/tax/individuals/topics/rrsps-related-plans/registered-retirement-savings-plan-rrsp.html)
