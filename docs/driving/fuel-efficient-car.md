# Fuel Efficient Car

- Small vehicle
- Continuous variable transmission (CVT) to allow an infinite number of gears
- Front-wheel drive
- Less horsepower
- Avoid unnecessary
  - Power windows, seats, mirrors, doors (they add weights but what?)
  - Remote car starter
  - Air conditioning without economy mode
  - Permanent roof rack
  - Turbocharged engines
- Fuel-efficient features
  - Aluminum wheels
  - Block heater
  - Cruise control
  - Navigation system
  - Removable roof rack
  - Sunroof with tilt function
  - Tachometer to show engine speed (only for manual transmission?)
  - Tilted windows
  - Trip computer
