# What is Boredom

悶唔係冇野做；悶係指 **所有可以做野，對自己嚟講都冇吸引力**；悶係指件事難以令自己集中，易想瞓。因為
咁，好多産品（例如 Facebook, Instagram, YouTube, Netflix）而家嘅目標係爭取人嘅碎片時間，想用戶喺悶嘅
時候用個産品。

心理學實驗發現，叫受試者喺一個房間 6-15 分鐘內除咗自己思考；或者禁一個帶嚟電擊嘅制，唔可以做其他野。
有 25% 女 67% 男會選擇禁制（**寧願身體痛都唔想悶**）。另外調查發現 95% 美國人喺過去 24h 有活動；只有
17% 會將多餘時間用嚟放鬆同思考

## Boredom Can Boost Creativity

有研究指出精神唔集中其實會提高創造力︰參加者被指定去做一啲悶嘅任務，當中最悶嘅係讀電話簿，之後研究員
叫參加者去諗一個膠杯有乜可以玩，讀電話簿嘅人係諗到最多玩法嘅一群。悶會令人想做改變，令自己更正面，甚
至有利他行為。悶會令人對自己人生有更多想法，回顧過去、展望將來（autobiographical planning）。我地更
可以反其道而行：因為思考呢件事係困難嘅，我地可以用悶嚟令自己覺得思考反而更有趣
