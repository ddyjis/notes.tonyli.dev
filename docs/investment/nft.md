# NFT

## Nature

NFT communities in 2021 are like clubhouses. You need to pay a lot for a token to join one and grant
you access to the parties. Like cryptos, only the largest ones have values to invest in so the
entrance fee is high as an investment.

## Value

NFTs are non-fugitive. 99% of them are **really** pointless, valueless ownership to digital images.
These NFTs are exposed to large market risk. If one day people lost faith in investing NFT market
(not NFT itself), no one would like to buy any of these because they are non-fugitive and non-liquid
in nature.
