# Notes should be written using my own words

I should use my own words when I want to write down ideas, _even though it is just paraphrasing_.
Learning only happens when I spend efforts to give output. At least get something out from my brain
instead of just copy and paste.

One way of doing it could be to make use of my bilingual ability - if an external idea comes from an
English source, I should write in Cantonese, otherwise, write in English. Since these two languages
are very different, this can force myself to process the information in my mind.

Of course there is exception where this rule does not apply (e.g. code snippets, definitions)
