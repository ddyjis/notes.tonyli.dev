# Less is more

The goal of this Pensieve is to build a GTD and PKM system. It is for "Getting Things Done", not "Getting this Pensieve Done". The Pensieve does not need to be perfect (In fact, what is a "perfect" Pensieve?). The  main idea is to get the max outcome while spend less time on it. Organize the Pensieve with minimal time is preferred.
