# Notes are written in markdown

Markdown files does not depend on any system or platform. Notes written in markdown can be migrated
easily, hence they are future-proof. Notes are also text-based so they can be tracked by version
control systems.

Being future-proof is important. Suppose I have a set of notes stored in a note hosting platform. If
there is a backward incompatible upgrade; or if that platform is gone / acquired by another company
and is somehow taken down later, I will have trouble moving all my data away. This happens a lot
especially the big-tech like to acquire popular services to maintain their competitiveness.
