# Currency

## Norbert's Gambit

This is a technique to minimize the cost of transferring between USD and CAD

If you have an IB account, skip this.

There is an ETF called [Horizons US Dollar Currency ETF](http://www.horizonsetfs.com/horizons/media/pdfs/productsheets/DLR-Product-Sheet.pdf) which has 2 tickers - [`DLR.TO`](https://www.horizonsetfs.com/etf/DLR) (in CAD) and [`DLR.U.TO`](https://www.horizonsetfs.com/etf/DLR-U) (in USD). The 2 tickers can be inter-changed into each other with the same number of shares by journaling.

The cost for the currency conversion is the commission fee only, which should be smaller than the spread of actual `USD/CAD`
