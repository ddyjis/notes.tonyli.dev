# Active Learning Vs Passive Learning

We learn from books, articles, podcasts, videos, etc. These contents are prepared by others which means we follow the authors' flow. We process all these information **passively**. This does not help our brain internalizing the idea. Instead we need to **actively** restructure, break down to learn more effectively.
