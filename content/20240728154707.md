# Project Euler Problem 28

```python
def p28(x):
  "Assume x is odd"
  if x == 1: return 1
  total = 1
  pointer = 1
  for size in range(3, x + 1, 2):
    diff = size - 1
    total += 4 * pointer + 10 * diff
    pointer += 4 * diff
  return total
```
